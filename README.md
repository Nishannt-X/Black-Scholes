# G3-M3 Black-Scholes Options Pricing Engine

## Overview
The G3-M3 Black-Scholes Options Pricing Engine is designed to facilitate the calculation of options pricing using the Black-Scholes model, providing essential features and functionalities for both educational and practical applications.

## Features
- **Real-time Pricing:** Instant calculation of option prices based on current market data.
- **Customizable Inputs:** Allows users to input various parameters including stock price, strike price, volatility, and risk-free interest rate.
- **Graphical User Interface (GUI):** User-friendly interface for ease of interaction and visualization of results.
- **API Access:** Comprehensive API documentation for integration with other platforms.

## Installation
To install the Black-Scholes Options Pricing Engine, please follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Nishannt-X/Black-Scholes.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Black-Scholes
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To use the Black-Scholes Options Pricing Engine:
1. Start the application:
   ```bash
   npm start
   ```
2. Input the required parameters in the GUI.
3. Review the generated option price and other relevant metrics.
